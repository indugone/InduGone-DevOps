*HOW TO CREATE MANAGEMENT CONSOLE USER:
-=====================================
<Home folder of wildfly>-->bin
file name : add -user (windows batch file)
once you run the add user it will asking the what type of user do you want
a) management user (mgmt-users.properties)
b)application user(application-users.properties)
a)type a and click on enter
after it will ask us a username after giving username and after it will ask us password and re enter the password

Deployments: A deployment represents anything that can be deployed (ex: an application such as RAR or JBoss-specific deployment) into a server.
*you can use drag and drop to add new content or replace existing deployments simply drag one or several files onto the deployment with the same name, the deployment will be replaced, otherwise the deployment will be replaced, otherwise the deployment will be added. The deployments added by drag and drop will be enabled by default.
